sdgbc

September 2017 - May 2018

C++

SFML

wxWidgets

CMake

A Nintendo Game Boy & Game Boy Color emulator developed for Windows and Linux.
The project won the 2018 best Computer Science project prize at the University of Leicester.

It features:

  • Video, sound and joypad emulation.
  • A command-line interface for debugging programs.
  • Emulation controls for pausing and fast-forwarding gameplay on-demand.
  • The ability to import and export save files.
